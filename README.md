了解です！では、「調査隊がソウル市の公式窓口に問い合わせを行う」という、現実的な行動とサスペンスを組み合わせた章を書きます。
“市役所への問い合わせ”という一見地味な行動を、逆に不気味な緊張感に仕上げます。

⸻

タイトル：沈黙の窓口

戒厳令による不可解な地下鉄の異常――閉ざされたドア、座席の使用禁止。
その原因を探るため、調査隊はついに ソウル市公式への問い合わせ を決めた。

「地下鉄会社じゃなく、市公式に聞こう。あそこが答えないなら、それ自体が手がかりだ」
ヘジンの言葉に、皆は静かにうなずいた。

⸻

◆ ソウル市総合市民センターへ

夜の市庁駅に降り立った調査隊は、冷たいビル風を受けながら、市民センターの窓口へ向かった。
建物のガラス扉には、戒厳令を知らせる貼り紙が貼られ、外灯の白い光が反射していた。

自動ドアが静かに開くと、受付にはほんの数人の職員しかいない。

「いらっしゃいませ。お問い合わせ内容は…？」
女性職員の声は丁寧だが、どこか機械的な響きがあった。

ソンミンが深呼吸をして言う。
「地下鉄の座席使用禁止について、公式の理由を教えてほしいんです」

職員は一瞬だけ指を止めた。
その短い静寂が、妙に重く感じられた。

⸻

◆ 不自然な返答

「確認いたしますので、お待ちください」
職員は電話を取り、誰かに連絡する。しかし、その様子は奇妙だった。
	•	数秒ごとに周囲を見回す
	•	声が極端に小さい
	•	返答を聞いても表情が動かない

ヘジンは気づいた。
「……なんだか、普通じゃない」

十分ほど経ち、職員は席に戻った。
そして、決められたセリフのように淡々と言った。

「地下鉄の座席使用禁止につきましては、現在、上からの指示により回答を控えております。」

「上から？」
ジフンが尋ねても、職員はそれ以上答えない。

「市役所の判断ではないんですか？」
ヘジンがさらに踏み込む。

職員は少しだけ目を伏せ、低い声で言った。

「…申し訳ありません。お問い合わせは、控えるように…とのことです。」

その瞬間、調査隊の背筋にひやりとしたものが走った。
“誰か”が、市職員までも沈黙させている。

⸻

◆ 監視される気配

市民センターを出ると、ソンミンのスマートフォンに不審な通知が入った。

『公的機関への不必要な問い合わせは控えてください』

差出人不明。
返信もできない。
まるでどこかで話を聞かれていたかのようだった。

ヘジンは夜空を見上げた。
ソウル市庁周辺の巨大ビルの窓が黒く沈み、まるで街全体が何かを隠しているように見える。

「……これは、市のレベルじゃない。もっと上だ」

ジフンが呟いた。
「“上”って誰だよ？」

その答えを知る者は、まだ誰もいなかった。

だが、調査隊は確信した――
この戒厳令、ただの政治判断ではない。何かが動いている。
