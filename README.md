九龍城砦、政府に見つかる瞬間（香港語入り）

朝の薄明かり。九龍城砦の狭い路地には、洗濯物が天井まで張り巡らされ、空気は煙草と煮炊きの匂いで満ちていた。鉄製の梯子や木の通路を伝って、子どもたちが笑い声を響かせながら屋根から屋根へ飛び移る。

その日、いつもと違う低い羽音が上空をかすめた。ヘリコプターだ。住民たちは顔を上げ、目を細める。窓の向こうに、白い文字が書かれた書類を手にした男性たちが見えた。香港政府の職員だった。

「喂，你哋睇下！」（ねえ、見てみろよ！）
中庭で猫を追いかけていた老人がつぶやく。住民は普段、行政をあまり気にせず暮らしていたが、ヘリのカメラが屋根を撮影していることに、さすがに気づかないわけにはいかなかった。

ヘリの影が屋根に落ちると、細い路地にパトロール隊が降り立った。役人たちは測量機材やノート、カメラを抱えて進む。狭い階段を上るたびに、住民たちは息を潜め、ひそひそと情報を交換する。

「影到相喇！」（写真撮られてるぞ！）
若い女性が隠しながら階段の影から叫ぶ。
「快啲躲埋啲細路！」（子どもを隠せ！）
もう一人の男が、屋根裏の隙間に小さな子どもをかくまった。

その後、調査チームは各建物を順に確認し、屋根裏や隠し部屋の数をメモした。鉄パイプと木材で即席に組まれた梯子、壁の隙間に置かれた簡易水槽、カーテン代わりの布――そこに住む人々の工夫が赤裸々に記録されていく。

数時間後、ヘリが去ると、住民たちは再び日常に戻る。しかし、誰もが胸の奥に「自分たちの街はもう秘密ではない」という感覚を抱えていた。官庁の記録には、次のように記されることになる：

「無秩序に建てられた住宅群。法律・規制の届かぬ区域。人口密集度極めて高し。」

そして、壁には手書きの看板がひとつ残される：

唔好亂入！(入るな！)

この瞬間から、九龍城砦は単なる迷宮ではなく、政府に「公式に認識された存在」として歴史に刻まれることになる。
