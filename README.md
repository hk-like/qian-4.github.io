🇹🇼 台湾・台北地下鉄で無差別刃物襲撃、3人死亡・25人負傷

概要（2014年5月21日発信）
台湾の首都・台北の地下鉄車内で、男が乗客に対して突然刃物で襲いかかる無差別殺傷事件が発生しました。3人が死亡し、25人が負傷しました。容疑者の男（21歳の学生）はその場で逮捕されましたが、動機は明らかになっていません。台北の地下鉄で死者が出たのは1996年の開業以来初めてのことです。 ￼

詳細
	•	事件発生は 5月21日午後4時頃（日本時間午後5時）。襲撃は地下鉄の列車内で突然始まり、乗客数百人が逃げ惑いました。 ￼
	•	現場では悲鳴が上がり、床には血が広がるなどの状況が伝えられています。 ￼
	•	台北地下鉄は日常的に多くの人が利用しており、このような重大事件は非常に稀です。 ￼
